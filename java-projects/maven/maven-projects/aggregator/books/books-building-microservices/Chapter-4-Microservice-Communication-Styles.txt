------------------------------------------------------------------------------------------------------------------------
Synchronous Blocking
------------------------------------------------------------------------------------------------------------------------
Order Processor -> Loyalty Service
If Loyalty Service is down, Order Processor is blocked until it comes back up.
When we have multiple synchronous calls, the chances of failure increase, and the user experience degrades.

------------------------------------------------------------------------------------------------------------------------
Order Processor -> Payment Service -> Fraud Detection Service -> Customer Service
------------------------------------------------------------------------------------------------------------------------
Resource Contention occurs when multiple services are trying to access the same resource, such as a database or an API. This can lead to performance issues and increased latency.
